How to Integrate Jewelry V0.12 Web Assembly into Your Project

Follow these steps to successfully integrate our Product WASM into your application.

Step 1: Include WASM Runner Library

Add the wasmrunner.min.js library to your project. Include it in your HTML file like so:

<script src="path/to/wasmrunner.min.js"></script>

Step 2: Load the WASM Module

Initialize the WASM Runner with the module ID and URL:

var wasmRunner = new CoherentGlobal.WasmRunner({
    id: "your_module_id",
    url: "path/to/your_module.wasm",
});

Step 3: Prepare Your Payload

Construct the payload object with your request data and metadata.

const payload = {
    request_data: {
        // Your request data here
    },
    request_meta: {
        // Your request metadata here
    }
};

Step 4: Execute the WASM Function

Use the .execute() method to submit your payload to the WASM module.

async function submitData() {
    try {
        const response = await wasmRunner.execute(payload);
        console.log("WASM Execution Response:", response);
    } catch (error) {
        console.error("Error executing WASM module:", error);
    }
}

Step 5: Handle the Response

Process the response from the WASM module as needed for your application.

Step 6: Error Handling

Implement error handling for initialization and execution phases of the WASM module.
